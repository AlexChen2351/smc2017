Particle Metropolis-Hastings algorithm for inference of the parameters in a stochastic volitlity model. This folder covers some alternative formulations of the stochastic volitility model, and also contains the original code by Johan Dahlin (cf. infra) for reference.

Reference: J. Dahlin and T. B. Sch¨on, “Getting Started with Particle Metropolis-Hastings for
Inference in Nonlinear Dynamical Models,” ArXiv e-prints, Nov. 2015.

Available from: https://arxiv.org/pdf/1511.01707.pdf
